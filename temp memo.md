#### radio
```html
<input type="radio" name="r" value="a">A
<input type="radio" name="r" value="b" checked>B
<input type="radio" name="r" value="c">C
```
radio: name 속성값이 같은 값들 중 하나만 선택
value: 무엇을 선택했는지 구분하기 위한 값
checked: B가 기본 선택되어있도록 함
