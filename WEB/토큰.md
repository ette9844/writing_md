### JAVA Tokenizer
```java
String s = "아버지가::방에:들어:가신다";

StringTokenizer st = new StringTokenizer(s, ":");
while(st.hasMoreTokens()){
	String t = st.nextToken();
	System.out.println(t);
}
```
>아버지가
>방에
>들어
>가신다
>
>빈 문자열은 무시

빈 문자열도 포함하기 위해서는 split 사용
```java
String[] arr = s.split(5, ":"); // 무조건 5개
```
