## 최대공약수 문제
정답이 분수로 나왔을 때, 이를 기약분수로 출력해라. 등에 최대공약수가 사용된다.

## 최대공약수
Greatest Common Divisor (GCD)

최대공약수를 구하는 가장 쉬운 방법은 2 부터 min(A, B)까지 모든 정수로 나누어 보는 방법
```java
int g = 1;
for(int i=2; i<=min(a, b); i++){
	if(a % i == 0 && b % i == 0){
		g = i;
	}
}
```
